f-qr-fixer
=========

Using a file given in format of an NxN matrix where x or X is black . is white and * is unknown

For example an fqr input where left side of qr code is destroyed:

```
****************XX....XXXXXXX
*****************.XXX.X.....X
****************XX.XX.X.XXX.X
****************X.X...X.XXX.X
*****************XX.X.X.XXX.X
*****************.X.X.X.....X
****************X.X.X.XXXXXXX
****************X...X........
*****************XX.XX.XXXXX.
****************XX.X.X.X....X
*****************X.X.X..XXXX.
*****************XX..X.XXXXXX
******************.X...XXX...
******************..X.XX.X.X.
******************...XXXXXXX.
******************..XX...X.X.
******************..XX.X..X.X
******************.XXX...X...
******************..X...X.X.X
******************...X..X..XX
******************.XXXXXX.XXX
******************..X...X...X
******************XXX.X.X.X..
******************XXX...X....
******************XXXXXXX.X.X
******************.X.X.X.XX..
******************.XXXX...XXX
******************XX...XX.X.X
******************.XX.XXXX.XX
```
